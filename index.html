Features

Everything you need.
Nothing you don't.

Select & translate

Select text in any app — Xcode, Safari, Slack, PDFs — press ⌥⌘T and you're done. Works everywhere, system-wide. Not your thing? Remap it to any combo in Settings.

Auto-flip

Writing in Spanish but need it in English? SwifT detects the language direction automatically and flips when needed. Bidirectional, always.

Powered by Apple

Uses Apple's on-device Translation framework on macOS 26+. Falls back to Google Translate gracefully — no API key, no setup, no account.

Stays out of your way

Lives quietly in the menu bar. No Dock icon, no onboarding screens, no subscriptions. Set it up once, forget about it. Launches at login.
